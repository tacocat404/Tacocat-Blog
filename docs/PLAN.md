# PLAN — 픽셀아트 포트폴리오 블로그 기획안

_최종 수정: 2026-06-30_

## 1. 목적
개인 **포트폴리오 · 수상내역 · 활동내역 · 공부기록**을 정리하는 개인 블로그.
운영자는 **비개발자** → 코드는 Claude가 전담, 운영자는 콘텐츠(글)만 작성.
추후 **Claude 연동 자동화**(마크다운 작성 → 자동 발행)로 확장.

## 2. 확정된 결정 (이전 기획에서 유지)
- 콘텐츠 카테고리: 포트폴리오 / 수상 / 활동 / 공부기록 / About
- 언어: 한·영 이중언어 (i18n)
- 도메인: `404Tacocat.com` (Vercel 자동 배포)
- 스택 후보: **Astro + Markdown 콘텐츠** (정적 사이트, 비개발자 운영에 적합)

## 3. 변경된 결정
- **디자인: 로우폴리 3D(Three.js) → 픽셀아트(2D) 로 전환** ✅
  - 사유: 사용자가 픽셀아트 풍을 선호. 3D보다 가볍고, 비개발자가 콘텐츠 추가해도 깨지지 않으며, 자동화와 궁합이 좋음.

## 4. 디자인 컨셉 확정: 레트로 데스크톱 OS (픽셀) ✅
- **컨셉**: 옛날 운영체제 바탕화면처럼 보이는 픽셀 블로그.
  - 바탕화면 **아이콘 = 섹션**(포트폴리오/수상/활동/공부기록/About)
  - 아이콘 클릭 → **창(window)** 이 열리며 글 목록·본문 표시
  - 하단 작업표시줄(taskbar) + START 버튼 = 전체 메뉴(모바일/접근성 폴백)
- **세부 확정**:
  - OS 감성: 커스텀 코지 레트로 (Win95 구조 + 따뜻한 톤) ✅
  - 팔레트: 크림 `#f5ecd9` / 피치 `#e9a86a` / 세이지 `#a7be9e` / 스카이 `#bbd8e2` / 코코아 `#8a6f55` ✅
  - 마스코트: Claude Code 마스코트(오렌지 스파크) 픽셀 스프라이트, 동작 없이 앉아만 있음 ✅
  - 첫 화면: 짧은 부팅 화면 → 바탕화면 ✅
  - 모바일: 세로 목록(앱 서랍) 레이아웃으로 전환 ✅

## 5. 단계
1. 기획 ✅
2. **구현** ← 현재 단계 (뼈대·디자인·예시 콘텐츠 완료 / 실제 콘텐츠 대기)
3. 배포 (Vercel + 도메인 연결)
4. 자동화 (Claude로 글 생성 → 발행 파이프라인)

## 6. 실행 방법 (참고)
- 개발 미리보기: `npm run dev` → 브라우저에서 http://localhost:4321
- 빌드 확인: `npm run build`
- 콘텐츠 추가법: `docs/CONTENT-GUIDE.md` 참고
