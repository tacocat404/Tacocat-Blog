# 콘텐츠 작성 가이드 (비개발자용)

이 블로그는 **글 한 개 = 파일 한 개**입니다. 코드를 몰라도 양식만 채우면 돼요.

## 파일은 어디에 있나요?
`src/content/` 폴더 아래에 섹션별로 나뉘어 있습니다.

| 섹션 | 폴더 |
|---|---|
| 포트폴리오 | `src/content/portfolio/` |
| 수상 | `src/content/awards/` |
| 활동 | `src/content/activities/` |
| 공부기록 | `src/content/study/` |

새 글을 쓰려면 그 폴더 안의 `example-...md` 파일을 복사해서 이름만 바꾸고 내용을 고치면 됩니다.

## 파일 양식
파일 맨 위 `---` 와 `---` 사이는 **정보 칸**, 그 아래는 **본문**입니다.

```markdown
---
title: 제목
period: 2025.03 – 2025.05      # 기간 (포트폴리오/활동)
summary: 한 줄 요약
tech: ["도구1", "도구2"]        # 포트폴리오 전용: 사용한 기술
category: "인공지능"            # 공부기록 전용: 분류 (인공지능/알고리즘/기타 또는 새 이름)
series: "시리즈명"              # 공부기록 전용: 시리즈로 묶을 때
---

여기에 본문을 자유롭게 씁니다.
- 줄 앞에 - 를 붙이면 목록이 됩니다.
## 소제목 도 쓸 수 있어요.
```

> ⚠️ 날짜는 꼭 따옴표로! `date: "2025.11"` (따옴표 없으면 숫자로 오해해서 오류가 나요.)

## 미리보기
1. 터미널에서 `npm run dev` 실행
2. 브라우저에서 http://localhost:4321 접속
3. 파일을 저장하면 화면이 자동으로 바뀝니다.
